# 🔍 File Search Tool

A Python command-line tool that searches for files and text within files.

## Features

- Search for files by keyword
- Search inside file contents
- Filters file types (e.g. .txt, .py, .md)
- Skips unnecessary folders (e.g. __pycache__, .git)

## How to Run

```bash
python3 file_search.py
