# 🧠 Smart File Manager (Python CLI Tool)

A command-line Python tool that automates file organisation and file searching.

---

## 📂 Features

### 1. File Organiser
- Organises files into folders:
  - Images
  - Documents
  - Others
- Uses file extensions to sort files automatically

### 2. File Search
- Searches for keywords in:
  - File names
  - File content
- Supports multiple folders
- Skips system folders like `__pycache__`

### 3. Search Results
- Displays matches in terminal
- Saves results to `search_results.txt`

---
